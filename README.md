# Selenium POM Automation Framework

This project demonstrates automation testing using the Page Object Model (POM) design pattern with Selenium WebDriver.

## Tech Stack
- Java
- Selenium WebDriver
- Maven
- Page Object Model (POM)

## Automated Websites
- Amazon
- Flipkart

## Features
- Reusable Page classes
- Clean and maintainable framework
- Real-time automation scenarios

## How to Run
1. Clone the repository
2. Import as Maven project
3. Run test classes
